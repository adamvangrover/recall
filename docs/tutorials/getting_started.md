# Getting Started

Welcome to the Recall System! This tutorial will guide you through your first steps.

1.  **Start the system:** Run the main executable `recall`.
2.  **Add Memories:** Try logging a memory using `recall add "I had a great meeting today."`.
3.  **Search Memories:** Find it again using `recall search "meeting"`.
4.  **Advanced Links:** Try linking memories together in the Knowledge Graph with `recall link <id1> <id2>`.

See the How-To guides for more specific tasks.
