# Getting Started

Welcome to the Recall System! This tutorial will guide you through your first steps with our interactive CLI.

1.  **Start the system:** You can view all available commands by running `recall --help`.
2.  **Add Memories:** Try logging a memory using `recall add "I had a great meeting today."`. You will see a green success panel and a Cyan ID.
3.  **Search Memories:** Find it again using `recall search "meeting"`. The CLI will return a formatted table highlighting matching IDs, the similarity score, and the text.
4.  **Advanced Links:** Try linking memories together in the Knowledge Graph with `recall link <id1> <id2>`.
5.  **Storage Optimization:** Try running `recall recall optimize` to view a panel breaking down potential savings for cold memories.

See the How-To guides for more specific tasks.
